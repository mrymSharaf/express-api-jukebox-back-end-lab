# express-api-jukebox-back-end-lab

## Routes

| HTTP Method | Route  | Description           |             
|-------------|--------|-----------------------|
| POST        | /new   | Create a new track    | 
| GET         | /      | Get all tracks        | 
| GET         | /:id   | Get track details     | 
| PUT         | /:id   | Update a track        | 
| DELETE      | /:id   | Delete a track        |
